Mengenal Kerangka Web

HTML, CSS, Javascript dan Tailwind
HTML

Kerangka dasar dari sebuah halaman web. HTML itu ibarat kerangka rumah, yang menahan segala sesuatu yang ada di dalamnya.

CSS

Digunakan untuk mengatur tampilan dan gaya dari halaman web. Ini merupakan cat dan dekorasi dari rumah kita.

JavaScript

Bahasa pemrograman yang digunakan untuk membuat halaman web menjadi interaktif dan dinamis. Ini adalah sistem kelistrikan dan mekanisme yang membuat rumah kita hidup.

Tailwind CSS

Framework CSS yang memudahkan kita dalam mendesain halaman web dengan menggunakan kelas-kelas yang sudah disediakan.

Membuat Web Sederhana!

Nah kita udah tau nih apa aja pondasi dan kerangka yang ada dalam sebuah web. Dengan semua itu, kita udah bisa nih buat web sederhana menggunakan informasi tadi.

Yang pertama yang harus kalian lakukan saat membuka VS Code kalian, kalian membuat file HTML bernama "index.html". For starter, kalian bisa menggunakan shortcut yang sudah ada di VS Code yaitu dengan menulis "!" lalu tekan "Tab". Dengan ini, kalian udah punya kerangka dasar dari sebuah web. Selanjutnya, kalian bisa menambahkan tag-tag HTML sesuai kebutuhan kalian.

"Sudah kakk, terus jelasin dong kak cara pengimplementasian CSS dan JavaScript"

Nah, untuk mengimplementasi CSS pada HTML kalian, kalian bisa menambahkan tag <link> di dalam tag <head> di file HTML kalian. CSS sudah diimplementasi! kalian udah bisa nih menambahkan style pada web kalian!

Selanjutnya ada JavaScript, kalian bisa menambahkan tag <script> sebelum tag penutup </body>. Dengan ini, kalian udah bisa nih membuat web kalian lebih interaktif!

"Horeee! Aku udah bisa bikin web sederhana nih, makasih kakak!"
